Pathfind Algorithm Visualizer
This project is designed to visualize pathfinding algorithms. User can design a maze and it will find a path from point A to point B. Users can select sorting algorithm and animation speed. Also users can create their own unique mazes by simply clicking and dragging. Black tiles drawn represent obstacles and pathfinding algorithms will go around created obstacles to find optimal path.
Pathfinding algorithms implemented are as follows:
  • Depth First Search
  • Breadth First Search
  • Djikstra's Algorithm
  • A*
